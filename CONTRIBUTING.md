# Contributing

This repository is currently a private staging repository.

## Scope of contributions

Only contribute material that is safe to publish or keep privately under the repository owner's control. Do not add:
- raw course submissions
- assignment instructions
- autograder files
- instructor tests
- starter code without confirmed redistribution rights
- notebooks containing student identifiers, credentials, or restricted data

## Implementation expectations

For each subproject:
- prefer a clean, independently written implementation
- keep documentation specific to the actual code present
- include small reproducible examples when redistribution is allowed
- separate verified behavior from planned work

## Before opening a change

Confirm:
- ownership and collaborator attribution are documented
- publication permission is settled for any group-derived material
- no secrets or personal identifiers are included
- restricted datasets are replaced with schema-only or synthetic examples when needed

## Validation

At the current repository state, validation evidence shows formatting and security tools ran successfully, but no Python source files or tests are present.
