# Computer Vision

Computer-vision notebooks covering segmentation, panorama stitching, 3D reconstruction, and stereo depth.

**Technologies:** Python · Jupyter · NumPy · OpenCV

## Highlights

- Color segmentation using probabilistic modeling.
- Feature matching and panorama construction.
- Two-view reconstruction and stereo depth estimation.

## Getting started

1. Open the selected notebook in JupyterLab or Google Colab.
2. Install the libraries imported by that notebook.
3. Add any required local images or datasets using the documented relative paths.
